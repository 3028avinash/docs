Explore by use case

Redis can be used as a database, cache, streaming engine, message broker, and more.

  • Data structure store

    Redis stands for Remote Dictionary Server. It allows using same data types as in your local programming environment but on the side of Redis.

    Quick start guide →

  • Document database

    Leverage JSON and its flexible data model to use Redis as a document database. Model, search, and query your data efficiently without needing an additional cache.

    Quick start guide →

  • Vector database

    Develop AI-powered applications by querying vector embeddings. You can use Redis as a vector database to implement recommendation systems, Retrieval Augmented Generation (RAG), LLM semantic caching, and more.

    Quick start guide →
